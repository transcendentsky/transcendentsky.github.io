用 GitHub Pages 重启个人网站

这是新模板中的第一篇文章，用来说明这个站点的结构和后续维护方式。

这个网站现在采用无构建依赖的静态 HTML 模板，适合放在 transcendentsky.github.io 仓库中直接托管。首页展示个人信息和精选入口，简历页集中维护正式履历，博客页保留新文章和旧归档。

如何新增文章

复制 blog/hello-github-pages/ 目录，改成新的文章路径，例如 blog/my-new-post/，然后修改其中的 index.html。最后在 blog/index.html 和首页的最新文章区域加上链接。

如何更新简历

编辑 resume/index.html 中的教育经历、项目经历、技能和联系方式即可。当前页面已经预留了常见简历区块。

旧文章

旧 Hexo 文章仍保留在 2018/、archives/、tags/ 等目录中。后续可以逐篇迁移，也可以继续作为归档入口保留。
